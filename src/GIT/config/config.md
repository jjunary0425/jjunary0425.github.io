# Git Config

## git config 확인
```
git config --list
```

## git config 설정
- 설정
```
git config --global user.name "RimaPa"
git config --global user.email "jjunary0425@gmail.com"
```
- 결과 확인
```
git config --list

credential.helper=osxkeychain
init.defaultbranch=main
user.name=RimaPa
user.email=jjunary0425@gmail.com
```
